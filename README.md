# DBT Skills Helper

An interactive web-based tool to help navigate and apply DBT (Dialectical Behavior Therapy) skills.

## Features

- Three-category triage system for different emotional states
- 13 skill infographics with step-by-step guidance
- Hand-drawn aesthetic with calming design
- Mobile-optimized and touch-friendly
- Works offline once loaded

## Categories

1. **I don't feel good / Something feels off** - Distress tolerance and emotion regulation
2. **My thoughts are racing/spiraling** - Mindfulness and cognitive techniques
3. **I'm stuck on a thought/urge** - OCD-specific and acceptance-based skills

## Usage

Simply open the site and select the category that matches your current experience. Then choose a skill to view the detailed infographic guide.

## Skills Included

- TIPP
- Sensory Soothing
- Distract Yourself
- Opposite Action
- Mindfulness
- Ask for Help
- Thought Observation
- Reframe Your Thoughts
- Urge Surfing
- Acceptance
- OCD Diffusion
- ERP (Exposure Response Prevention)
- Mindfulness for OCD
